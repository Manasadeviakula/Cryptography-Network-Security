# Cryptography & Network Security Lab Programs

**This repository contains lab programs taught by TMC Uday, Professor, CSE.**  

The repository includes implementations of classical and modern cryptography algorithms using C, Java, and JavaScript. These programs are part of the practical curriculum for the Cryptography and Network Security course.

---

## 📚 Lab Programs

### C Programs

1. **XOR Hello World**  
   Write a C program that contains a string (`char *`) with value `'Hello World'`. XOR each character in this string with `0` and display the result.

2. **AND/XOR Hello World**  
   Write a C program that contains a string (`char *`) with value `'Hello World'`. AND or XOR each character in this string with `127` and display the result.

---

### Java Programs

3. **Classical Ciphers**  
   Perform encryption and decryption using the following algorithms:  
   - Caesar Cipher  
   - Substitution Cipher  
   - Hill Cipher

4. **DES Algorithm**  
   Implement the DES (Data Encryption Standard) algorithm logic.

5. **Blowfish Algorithm**  
   Implement Blowfish encryption and decryption logic (C/Java).

6. **Rijndael Algorithm**  
   Implement Rijndael encryption logic (C/Java).

7. **Blowfish with Java Cryptography**  
   Encrypt the text `"Hello World"` using Blowfish. Create your own key using Java keytool.

8. **RSA Algorithm**  
   Implement RSA encryption and decryption logic in Java.

---

### Web & JavaScript

9. **Diffie-Hellman Key Exchange**  
   Implement the Diffie-Hellman Key Exchange mechanism using HTML and JavaScript.  
   Consider the end user as one party (Alice) and the JavaScript application as the other party (Bob).
